# ⏱️ Stopwatch with Lap Recorder  

### 📌 Description  
A simple **stopwatch application** that allows users to start, pause, reset, and record **lap times**.  
Designed with a minimal, beginner-friendly UI to practice JavaScript DOM manipulation.  

### 💻 Features  
- Start, pause, reset stopwatch  
- Record multiple lap times  
- Clean and user-friendly interface  

### 🛠️ Tech Stack  
- **HTML**  
- **CSS**  
- **JavaScript**  

### ⚡ How to Run Locally  
```bash
# Clone the repository
git clone https://github.com/Pranav-Savant/Stopwatch.git
cd Stopwatch
open index.html   # or double-click index.html
